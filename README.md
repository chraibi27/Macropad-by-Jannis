# Macropad by Jannis

A fully custom-built 9-key macropad based on the **Seeed XIAO RP2040** – including a self-designed PCB, 3D-printed enclosure, and custom firmware using KMK.  
This project is based on a complete DIY blueprint and has been individually expanded and refined.

---

## ✨ Features

- 9 mechanical MX switches  
- 2× SK6812 MINI RGB LEDs  
- Seeed XIAO RP2040 (soldered or socketed)  
- Custom PCB designed in KiCad  
- 3D-printed enclosure (Fusion360)  
- Firmware powered by KMK (Python)  
- Supports macros, shortcuts, and RGB lighting
